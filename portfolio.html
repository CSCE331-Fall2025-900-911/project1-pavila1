About Me & My Projects

Below are some of the technical projects I've worked on. Each project presented a unique challenge and provided a valuable learning experience in software development, hardware integration, and problem-solving.

Real-Time Weather Data Logger & Web Visualizer

This project involved creating a full end-to-end IoT system. I developed robust firmware in C++ for an ESP32 microcontroller to poll a BME280 sensor for real-time temperature, humidity, and pressure data. The device then publishes this data over Wi-Fi using the MQTT protocol. A Python backend script subscribes to the data feed, parses it, and logs it for analysis.
